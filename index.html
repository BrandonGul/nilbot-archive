NILBOT

This is an archive site for nilbot

While active, NILBOT was a fully functional Twitch moderation bot.

Using the API provided from Twitch the bot was able to have tight integration with all of fantasic features it provided.

Being built with great open source technologies such as: ExpressJS and TmiJS. NILBOT was able to achive a high level excellence. NILBOT also had a full backend infrastructure to support users and their respective Twitch accounts.


Unfortunately NILBOT was decommissioned in 2021 due to a lack of users to sustain the project.
